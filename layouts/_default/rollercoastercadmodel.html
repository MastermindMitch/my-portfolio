What:A passion project to create a model of the first ride I ever worked at using only a 3D printer.

Process:Used Google Earth to record the height of the track every 30 feet. I also used it to stitch together top-down view images to create a CAD model of the layout.

3D Printing:Used vase mode to print a single continuous wall at 0.05mm layer height. This achieved incredibly smooth results.

Projects Future:Incredible work has been made, but the project is on hold until I have more time.

CAD:Fusion 360

3D Printer:Bambu X1C

The track is split into over 200 segments for easier printing.

High Quality

The track piece on the top is printed at a 0.05mm layer height and is very smooth to the human eye. The bottom one has a 0.16mm layer height. This is under a microscope.
